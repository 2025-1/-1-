찾아오는 방법

판암 IC 경로:
판암IC출구 → 옥천길[대전]방면으로 좌회전 → 옥천길 판암지구대 삼거리에서 용운동길 방면으로 우회전 → 용운동길 사거리에서 [대전역/대동오거리] 방면으로 좌회전 → 용운 주공아파트2단지 사거리에서 좌회전 → 백룡길 직진 → 대학로 네거리에서 직진 → 우송대학교 동 캠퍼스 직진 → 우송대학교 서 캠퍼스

고속버스터미널:
102번: 고속버스 터미널에서 길 건너 버스정류장 탑승 → 한국화장품 앞 → 가양4거리 → 우송 서 캠퍼스(동문)입구 하차

605번: 동광장역 탑승 → 대전역(중앙시장) → 대동역 → 우송예술회관 앞 하차
608번: 대전역 탑승 → 대동역 → 우송대삼거리 하차 → 도보 10분

103번: 대전역 탑승 → 대동역 → 우송예술회관 앞 하차
